# S.I.M.A.-Clasa

**S.I.M.A. (Simulator Interactiv al Maselor Astronomice)** is a planetary physics simulator written in **C/C++** using **raylib**.  
It provides an interactive environment to visualize and simulate gravitational interactions between celestial bodies, making it an ideal tool for learning, experimentation, and demonstration of classical mechanics and orbital physics.

---

## 🪐 Overview

S.I.M.A. simulates the motion of planets, stars, and other bodies influenced by gravity. Built with **raylib** for real-time graphics rendering, this project lets you:

- Visualize orbital dynamics
- Interact with bodies in simulation
- Observe energy and momentum in action
- Explore physics concepts like escape velocity, gravity wells, and conservation laws

Whether you’re a student, hobbyist, or developer interested in physics simulation or game programming with raylib, S.I.M.A. offers a clear and extensible foundation.

---

## 🚀 Key Features

- 🌍 **Planetary Physics Engine**  
  Numerical integration of Newtonian gravity between multiple bodies.

- 📊 **Interactive Visualization**  
  Real-time rendering of bodies and trajectories using raylib.

- 🧮 **Configurable Bodies**  
  Define mass, position, velocity, and other physics parameters.

- 🎮 **Simulation Controls**  
  Start, pause, reset, zoom.

- ✨ **Extensible Codebase**  
  Designed for readability and expansion: add forces, render styles, or UI enhancements.
