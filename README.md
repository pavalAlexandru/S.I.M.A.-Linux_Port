# S.I.M.A. - Enhanced Linux Port 🌌

Originally a passion project developed during my high school years, **S.I.M.A. (Simulator Interactiv al Maselor Astronomice)** is a planetary physics simulator written in **C++** using **raylib**. 

This repository houses the **enhanced Linux port** of the original application. It breathes new life into a personal favorite project by bringing native Linux support, under-the-hood improvements, and an even better interactive environment to visualize and simulate gravitational interactions between celestial bodies.

---

## 🪐 Overview

S.I.M.A. simulates the motion of planets, stars, and other bodies influenced by gravity. Built with **raylib** for real-time graphics rendering, this project lets you:

- Visualize complex orbital dynamics in real-time.
- Interact dynamically with bodies during the simulation.
- Observe energy, momentum, and classical mechanics in action.
- Explore physics concepts like escape velocity, gravity wells, and conservation laws.

Whether you’re a student, a physics enthusiast, or a developer interested in game programming with C++ and raylib, S.I.M.A. offers a clear, visual, and extensible sandbox.

---

## 🚀 Key Features

- 🐧 **Enhanced Linux Port** Upgraded and optimized from the original high school codebase to run natively and smoothly on Linux environments.

- 🌍 **Planetary Physics Engine** Accurate numerical integration of Newtonian gravity between multiple celestial bodies.

- 📊 **Interactive Visualization** Smooth, real-time rendering of bodies and orbital trajectories using the power of raylib.

- 🧮 **Configurable Bodies** Easily define and tweak mass, position, velocity, and other physical parameters to create custom solar systems.

- 🎮 **Simulation Controls** Full control over the simulation time: start, pause, reset, and zoom in/out to explore the sandbox.

- ✨ **Extensible Codebase** Designed for readability and future expansion. Feel free to fork and add new forces, rendering styles, or UI enhancements!
